# Dokan Management System Setup Guide

---

## 🛠️ Prerequisites

- **Composer**
- **Git**
- **VS Code**

---


## ⚙️ Project Configuration (run the terminal)

### **step 1: Git Pull**
```bash
git pull origin main
```

### **Step 2: Update Composer Dependencies**
```bash
composer update
```

### **Step 3: Serve the Project**
```bash
php artisan serve
```
